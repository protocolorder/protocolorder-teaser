Abstract

1. 概要

Hashranチェーンは、ゲーム内都市Hashranの経済活動、記録、検証、秩序維持を表現する架空の基盤台帳です。Hashranトークン（HASRNTK）は、Gas、基軸通貨、報酬資産、セキュリティ資源として機能し、戦闘、育成、取引、ウォレット管理を一つの意思決定体系へ接続します。

本仕様は、専門知識の説明ではなく、プレイヤーの判断とリスク管理をゲーム体験へ変換するための設計資料です。

Token Model

5. Hashranトークン（HASRNTK）

5.1 役割

Gasチェーン上の操作に必要な利用料。
基軸通貨ゲーム内の交換レート・評価の基準。
報酬資産戦闘勝利やブロック生成により獲得する基本報酬。
セキュリティ資源ネットワーク運用コストを担保し、スパム的行動を抑制。

5.2 獲得手段

獲得経路内容
戦闘勝利報酬経験値・ドロップと並ぶ基礎報酬として付与される。
ブロック生成（育成）Tierポイント獲得に加え、低確率エアドロップとして付与される。
取引ゲーム内取引所（CEX/DEX）を介した交換・流動性供給により獲得する。

5.3 消費手段

消費経路内容
送金ウォレット間移動、対プレイヤー/対NPC送金。
取引交換（スワップ）、清算、手数料。
契約オンチェーン処理、戦闘関連の特殊処理、経済スキル等の実行。
税・公課ゲーム内での課税イベントにおける納付。

5.4 価格と換金性

HASRNTKはゲーム内資産であり、現実世界での換金価値を前提としません。ゲーム内レートはバランス調整のために制御され、現実の市場価格を参照しません。

Economic Layer

6. 経済システムとの接続

6.1 メインウォレット / サブウォレット運用

メインウォレットは基軸資産の集約と長期保管を担います。サブウォレットは短期運用、戦闘、取引、レバレッジの実行口座として機能します。資金移動にはGasが発生し、反映まで時間がかかる場合があります。

6.2 取引所（CEX/DEX）と流動性

ゲーム内には取引所が存在し、通貨間の交換が可能です。ただし、交換の成立には流動性が必要であり、状況によっては希望レートで約定しません。この設計により、資産移動と戦闘準備に現実的な制約が生じます。

6.3 レバレッジ（証拠金）

戦闘開始時にレバレッジON/OFFを選択し、ON時は証拠金への投入量を決定します。戦闘中に証拠金を増減してレバレッジを調整する行為はできません。この制約により、事前の資金設計が戦術の一部となります。

Cultivation Layer

7. ブロック生成（育成）との接続

プレイヤーはブロック生成を通じてTierポイントを獲得し、成長・解放の進行度を管理します。ブロック生成の成果として、低確率でHASRNTKがエアドロップされます。

方針は「適当に育てても収穫できるが、丁寧に育てるほうが実りが多い」です。放置と能動プレイを両立させ、育成を攻略上の義務ではなく、帰還後の緩衝材として機能させます。

Security Model

8. セキュリティと不正対策

Gasによる摩擦スパム行動を抑止する。
遅延によるリスク管理資金移動の乱用を抑止し、判断ミスをコスト化する。
合意の重み過去改ざんを「理論上可能だが割に合わない」状態に置く。
可変合意（MCP）異常時に合意形態を遷移させ、破局を回避する。

Appendix / Oral Tradition

住民口伝版：Hashranの十二戒

分類：生活規範（簡易版／口伝）

第一戒

聖域は他人に見せるな

財布（聖域）は覗かせるな。見せた瞬間に、運は逃げる。

第二戒

魂の刻印は渡すな

署名の印は身体の一部。貸せば人生ごと持っていかれる。

第三戒

復活の呪文は唱えるな

十二の言葉は最後の最後まで封じよ。口にした回数だけ、呪いは近づく。

第四戒

供物なしに魔法を放つな

供物（触媒）を惜しむな。惜しめば、肝心な時に何も動かない。

第五戒

大金は奥へ、小銭は手元へ

大事なものは奥の聖域へ。今日使う分だけを手元の袋へ。

第六戒

移すなら早めに移せ

袋の移し替えには時間がかかる。戦いの前に慌てるな。

第七戒

1日に一回、祈りを天に届けよ

日が沈む前に、今日の出来事を束ね、天へ送れ。天は見ている。返事がなくても、送れ。

第八戒

祈りは一人で行うな

祈る者の隣に、見届ける者を置け。見届けがなければ、祈りは歪む。

第九戒

祈りの記録を残せ

祈った証は、地区の箱に収めよ。箱が空になれば、町は迷う。

第十戒

契りは軽々しく交わすな

誓約は精霊が勝手に執行する。破りたくなっても破れない。

第十一戒

安い近道を信じるな

甘い儲け話は帳尻が合わない。正門を通れ。

第十二戒

世界の記録を疑うな

記録は石積み。壊すのは簡単だが、積み直すのは現実的ではない。
